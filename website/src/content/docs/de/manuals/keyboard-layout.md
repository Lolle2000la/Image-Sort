---
title: Tastaturschema & Steuerung
description: Standard-Tastaturbelegung und Navigationsregeln für Media Sort.
---

Media Sort wurde von Grund auf für maximale Geschwindigkeit und Effizienz mit Ihrer Tastatur entwickelt. Diese Anleitung beschreibt die Standard-Navigationsregeln und Tastaturbelegungen.

## Raster-Navigation (Media Grid)

Verwenden Sie die folgenden Tasten, um im Medienraster (Miniaturansichten) zu navigieren:

| Aktion | Standard-Shortcut | Beschreibung |
| :--- | :--- | :--- |
| **Nach links / Zurück** | `Pfeiltaste Links` | Wählt die vorherige Mediendatei aus. |
| **Nach rechts / Vorwärts** | `Pfeiltaste Rechts` | Wählt die nächste Mediendatei aus. |
| **Auswahl öffnen** | `Eingabetaste (Enter)` | Öffnet den ausgewählten Ordner oder führt Doppelklick-Verhalten aus. |

---

## Ordnerbaum-Navigation

Verwenden Sie die Standardtasten WASD, um durch die Ordnerstruktur zu navigieren, sie aufzuklappen oder zuzuklappen:

| Aktion | Standard-Shortcut | Beschreibung |
| :--- | :--- | :--- |
| **Auswahl nach oben** | `W` | Bewegt den Cursor im Ordnerbaum nach oben. |
| **Auswahl nach unten** | `S` | Bewegt den Cursor im Ordnerbaum nach unten. |
| **Zuklappen / Links** | `A` | Klappt den ausgewählten Ordner zu oder springt zum übergeordneten Ordner. |
| **Aufklappen / Rechts** | `D` | Klappt den ausgewählten Ordner auf oder springt zum ersten Unterordner. |

---

## Sortier- & Dateiaktionen

Führen Sie Dateiverwaltungsaktionen sofort auf der aktiven Mediendatei aus:

| Aktion | Standard-Shortcut | Beschreibung |
| :--- | :--- | :--- |
| **In Ordner verschieben** | `Pfeiltaste Oben` | Verschiebt die ausgewählte Mediendatei in den aktiven Zielordner. |
| **In Ordner kopieren** | `Shift + Pfeiltaste Oben` | Kopiert die ausgewählte Mediendatei in den aktiven Zielordner. |
| **Löschen** | `Pfeiltaste Unten` | Verschiebt die ausgewählte Mediendatei in den System-Papierkorb. |
| **Umbenennen** | `R` | Öffnet den Dialog zum Umbenennen der aktuellen Mediendatei. |
| **Aktion rückgängig machen** | `Q` | Macht das letzte Verschieben, Kopieren, Umbenennen oder Löschen rückgängig. |
| **Aktion wiederholen** | `E` | Holt die rückgängig gemachte Aktion wieder zurück. |

## Programmoberfläche & Einstellungen

| Aktion | Standard-Shortcut | Beschreibung |
| :--- | :--- | :--- |
| **Ordnerauswahl öffnen** | `O` | Öffnet den nativen Systemdialog zur Auswahl eines neuen Quellverzeichnisses. |
| **Suchleiste fokussieren** | `I` | Setzt den Fokus in das Suchfeld, um Mediendateien zu filtern. |
| **Metadaten-Panel umschalten** | `M` | Blendet die rechte Metadaten-Leiste ein oder aus. |
| **Ordner erstellen** | `C` | Erstellt einen neuen Unterordner im aktiven Ordner. |

## Angepinnte Ordner

Angepinnte Ordner sind Sortierziele, die Tastenkombinationen zugewiesen sind:

| Aktion | Standard-Shortcut | Beschreibung |
| :--- | :--- | :--- |
| **In angepinnten Ordner 1-9 verschieben** | `Alt + 1` bis `Alt + 9` | Verschiebt die ausgewählte Mediendatei direkt in den angepinnten Zielordner. |
| **Ordnerauswahl zum Anpinnen öffnen** | `P` | Öffnet einen nativen Ordnerauswahldialog, um einen neuen Ordner zum Anpinnen auszuwählen. |
| **Ausgewählten Ordner anpinnen** | `F` | Pinnt den ausgewählten Ordner im Ordnerbaum an. |
| **Aktuellen Ordner entpinnen** | `U` | Entpinnt den aktuell aktiven Ordner. |
| **Pin nach oben verschieben** | `Strg + W` | Verschiebt den angepinnten Ordner in der Liste nach oben. |
| **Pin nach unten verschieben** | `Strg + S` | Verschiebt den angepinnten Ordner in der Liste nach unten. |

## Medienwiedergabe-Shortcuts

Diese Shortcuts sind **fest vorgegeben** und können nicht über den Einstellungsdialog geändert werden:

| Aktion | Standard-Shortcut | Beschreibung |
| :--- | :--- | :--- |
| **Pause/Fortsetzen umschalten** | `Leertaste` | Video-Wiedergabe pausieren oder fortsetzen. |
| **Medien Wiedergabe/Pause** | `MediaPlayPause` / `MediaPlay` / `MediaPause` | Video-Pause umschalten (Multimediatasten). |
| **Video stoppen** | `MediaStop` | Video-Wiedergabe stoppen. |
| **Nächstes Medium** | `MediaTrackNext` | Zur nächsten Mediendatei springen. |
| **Vorheriges Medium** | `MediaTrackPrevious` | Zur vorherigen Mediendatei springen. |
| **Lauter** | `AudioVolumeUp` | Lautstärke um 5 % erhöhen. |
| **Leiser** | `AudioVolumeDown` | Lautstärke um 5 % verringern. |
| **Stummschaltung umschalten** | `AudioVolumeMute` | Stummschaltung ein-/ausschalten. |
| **Anwendung beenden** | `Strg + Q` | Media Sort sofort beenden. |
