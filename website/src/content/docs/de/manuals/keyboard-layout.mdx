Media Sort wurde von Grund auf für maximale Geschwindigkeit und Effizienz mit Ihrer Tastatur entwickelt. Diese Anleitung beschreibt die Standard-Navigationsregeln und Tastaturbelegungen.

## Raster-Navigation (Media Grid)

Verwenden Sie die folgenden Tasten, um im Medienraster (Miniaturansichten) zu navigieren:

| Aktion | Standard-Shortcut | Beschreibung |
| :--- | :--- | :--- |
| **Nach links / Zurück** | <Kbd>Left</Kbd> | Wählt die vorherige Mediendatei aus. |
| **Nach rechts / Vorwärts** | <Kbd>Right</Kbd> | Wählt die nächste Mediendatei aus. |
| **Auswahl öffnen** | <Kbd>Enter</Kbd> | Öffnet den ausgewählten Ordner oder führt Doppelklick-Verhalten aus. |

---

## Ordnerbaum-Navigation

Verwenden Sie die Standardtasten WASD, um durch die Ordnerstruktur zu navigieren, sie aufzuklappen oder zuzuklappen:

| Aktion | Standard-Shortcut | Beschreibung |
| :--- | :--- | :--- |
| **Auswahl nach oben** | <Kbd>W</Kbd> | Bewegt den Cursor im Ordnerbaum nach oben. |
| **Auswahl nach unten** | <Kbd>S</Kbd> | Bewegt den Cursor im Ordnerbaum nach unten. |
| **Zuklappen / Links** | <Kbd>A</Kbd> | Klappt den ausgewählten Ordner zu oder springt zum übergeordneten Ordner. |
| **Aufklappen / Rechts** | <Kbd>D</Kbd> | Klappt den ausgewählten Ordner auf oder springt zum ersten Unterordner. |

---

## Sortier- & Dateiaktionen

Führen Sie Dateiverwaltungsaktionen sofort auf der aktiven Mediendatei aus:

| Aktion | Standard-Shortcut | Beschreibung |
| :--- | :--- | :--- |
| **In Ordner verschieben** | <Kbd>Up</Kbd> | Verschiebt die ausgewählte Mediendatei in den aktiven Zielordner. |
| **In Ordner kopieren** | <Kbd>Shift+Up</Kbd> | Kopiert die ausgewählte Mediendatei in den aktiven Zielordner. |
| **Löschen** | <Kbd>Down</Kbd> | Verschiebt die ausgewählte Mediendatei in den System-Papierkorb. |
| **Umbenennen** | <Kbd>R</Kbd> | Öffnet den Dialog zum Umbenennen der aktuellen Mediendatei. |
| **Aktion rückgängig machen** | <Kbd>Q</Kbd> | Macht das letzte Verschieben, Kopieren, Umbenennen oder Löschen rückgängig. |
| **Aktion wiederholen** | <Kbd>E</Kbd> | Holt die rückgängig gemachte Aktion wieder zurück. |

## Programmoberfläche & Einstellungen

| Aktion | Standard-Shortcut | Beschreibung |
| :--- | :--- | :--- |
| **Ordnerauswahl öffnen** | <Kbd>O</Kbd> | Öffnet den nativen Systemdialog zur Auswahl eines neuen Quellverzeichnisses. |
| **Suchleiste fokussieren** | <Kbd>I</Kbd> | Setzt den Fokus in das Suchfeld, um Mediendateien zu filtern. |
| **Metadaten-Panel umschalten** | <Kbd>M</Kbd> | Blendet die rechte Metadaten-Leiste ein oder aus. |
| **Ordner erstellen** | <Kbd>C</Kbd> | Erstellt einen neuen Unterordner im aktiven Ordner. |

## Angepinnte Ordner

Angepinnte Ordner sind Sortierziele, die Tastenkombinationen zugewiesen sind:

| Aktion | Standard-Shortcut | Beschreibung |
| :--- | :--- | :--- |
| **In angepinnten Ordner 1-9 verschieben** | <Kbd>Alt+1</Kbd> bis <Kbd>Alt+9</Kbd> | Verschiebt die ausgewählte Mediendatei direkt in den angepinnten Zielordner. |
| **Ordnerauswahl zum Anpinnen öffnen** | <Kbd>P</Kbd> | Öffnet einen nativen Ordnerauswahldialog, um einen neuen Ordner zum Anpinnen auszuwählen. |
| **Ausgewählten Ordner anpinnen** | <Kbd>F</Kbd> | Pinnt den ausgewählten Ordner im Ordnerbaum an. |
| **Aktuellen Ordner entpinnen** | <Kbd>U</Kbd> | Entpinnt den aktuell aktiven Ordner. |
| **Pin nach oben verschieben** | <Kbd>Ctrl+W</Kbd> | Verschiebt den angepinnten Ordner in der Liste nach oben. |
| **Pin nach unten verschieben** | <Kbd>Ctrl+S</Kbd> | Verschiebt den angepinnten Ordner in der Liste nach unten. |

## Medienwiedergabe-Shortcuts

Diese Shortcuts sind **fest vorgegeben** und können nicht über den Einstellungsdialog geändert werden:

| Aktion | Standard-Shortcut | Beschreibung |
| :--- | :--- | :--- |
| **Pause/Fortsetzen umschalten** | <Kbd>Space</Kbd> | Video-Wiedergabe pausieren oder fortsetzen. |
| **Medien Wiedergabe/Pause** | <Kbd>MediaPlayPause</Kbd> / <Kbd>MediaPlay</Kbd> / <Kbd>MediaPause</Kbd> | Video-Pause umschalten (Multimediatasten). |
| **Video stoppen** | <Kbd>MediaStop</Kbd> | Video-Wiedergabe stoppen. |
| **Nächstes Medium** | <Kbd>MediaTrackNext</Kbd> | Zur nächsten Mediendatei springen. |
| **Vorheriges Medium** | <Kbd>MediaTrackPrevious</Kbd> | Zur vorherigen Mediendatei springen. |
| **Lauter** | <Kbd>AudioVolumeUp</Kbd> | Lautstärke um 5 % erhöhen. |
| **Leiser** | <Kbd>AudioVolumeDown</Kbd> | Lautstärke um 5 % verringern. |
| **Stummschaltung umschalten** | <Kbd>AudioVolumeMute</Kbd> | Stummschaltung ein-/ausschalten. |
| **Anwendung beenden** | <Kbd>Ctrl+Q</Kbd> | Media Sort sofort beenden. |
