---
title: Aus dem Quellcode bauen
description: So bauen Sie Media Sort direkt aus dem Quellcode.
---

Wenn Sie Media Sort lieber selbst kompilieren möchten, benötigen Sie die Rust-Toolchain (Version **1.96** oder höher) und Clang (erforderlich, um Bindungen über `libmpv-sys` zu generieren). Media Sort benötigt außerdem `libmpv` zur Laufzeit für die Video-Wiedergabe.

---

## Build-Voraussetzungen

Installieren Sie vor dem Kompilieren die erforderlichen Compiler-Tools und Entwicklungsbibliotheken:

### Linux

#### Arch Linux / CachyOS

```bash
sudo pacman -S clang mpv
```

#### Debian / Ubuntu

```bash
sudo apt install clang libclang-dev libmpv-dev libasound2-dev libxkbcommon-dev libwayland-dev libx11-dev libvulkan-dev
```

### macOS

Installieren Sie die Xcode-Befehlszeilentools und Homebrew-Abhängigkeiten:

```bash
xcode-select --install
brew install mpv
```

### Windows

Installieren Sie die folgenden Voraussetzungen:

- [Rust-Toolchain](https://rustup.rs/) (1.96 oder höher)
- [Microsoft Visual Studio Build Tools](https://visualstudio.microsoft.com/visual-cpp-build-tools/) (für den MSVC-Linker)
- [LLVM / Clang](https://github.com/llvm/llvm-project/releases)
- mpv-Entwicklungsdateien über [vcpkg](https://github.com/microsoft/vcpkg):
    ```powershell
    vcpkg install mpv
    ```

---

## Projekt kompilieren

1. Klonen Sie das Projekt-Repository:

    ```bash
    git clone https://github.com/Lolle2000la/Image-Sort.git
    cd Image-Sort
    ```

2. Kompilieren Sie den Workspace mit Cargo:

    ```bash
    cargo build --release
    ```

3. Die kompilierte Binärdatei wird unter dem Zielverzeichnis generiert:
    ```bash
    ./target/release/media-sort-gui
    ```

---

## Laufzeitanforderungen

Nach dem Bau muss `libmpv` auf Ihrem System installiert sein, um die Video-Wiedergabe zu ermöglichen. Ohne `libmpv` funktioniert Media Sort weiterhin für Bilder und Audio, jedoch sind die interne Video- und GIF-Wiedergabe deaktiviert. Videodateien werden weiterhin erkannt und können durchsucht werden, allerdings ohne Miniaturansichten.
