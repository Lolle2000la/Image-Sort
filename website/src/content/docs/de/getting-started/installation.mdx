## Systemvoraussetzungen

Um Media Sort auszuführen, müssen die `mpv`-Demuxer-Bibliotheken (`libmpv`) auf Ihrem System installiert sein.

### Linux

#### Arch Linux / CachyOS
```bash
sudo pacman -S mpv
```

#### Debian / Ubuntu
```bash
sudo apt install libmpv-dev
```

#### Fedora
```bash
sudo dnf install mpv-libs-devel
```

### macOS

Stellen Sie sicher, dass Homebrew installiert ist, und führen Sie dann aus:
```bash
brew install mpv
```

### Windows
Es ist keine manuelle Einrichtung von Systembibliotheken erforderlich! Die vorkompilierten Windows-Installationsprogramme und ZIP-Pakete enthalten bereits die benötigte Bibliothek `libmpv-2.dll`.
