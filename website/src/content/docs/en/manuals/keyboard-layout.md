---
title: Keyboard Layout Schema
description: Default keybindings and navigation rules for Media Sort.
---

Media Sort is designed from the ground up for maximum speed and efficiency using only your keyboard. This guide details the default navigational rules and keyboard mappings.

## Grid Navigation

Use the following keys to navigate the media grid (thumbnails):

| Action | Default Keybinding | Description |
| :--- | :--- | :--- |
| **Go Left / Previous** | `Left Arrow` | Select the previous media file. |
| **Go Right / Next** | `Right Arrow` | Select the next media file. |
| **Open Selected** | `Enter` | Open the selected folder or double-click behavior. |

---

## Folder Tree Navigation

Use the standard WASD keys to navigate, expand, and collapse the directory tree structure:

| Action | Default Keybinding | Description |
| :--- | :--- | :--- |
| **Move Selection Up** | `W` | Move the cursor up the folder tree. |
| **Move Selection Down** | `S` | Move the cursor down the folder tree. |
| **Collapse / Left** | `A` | Collapse the selected folder node or go to the parent node. |
| **Expand / Right** | `D` | Expand the selected folder node or go to its first child. |

---

## Sorting & Action Mappings

Perform file management tasks on the active media file instantly:

| Action | Default Keybinding | Description |
| :--- | :--- | :--- |
| **Move to Folder** | `Up Arrow` | Move the currently selected media file to the active folder. |
| **Copy to Folder** | `Shift + Up Arrow` | Copy the currently selected media file to the active folder. |
| **Delete** | `Down Arrow` | Move the currently selected media file to the system Trash. |
| **Rename** | `R` | Trigger the rename dialog for the current media file. |
| **Undo Last Action** | `Q` | Undo the last move, copy, rename, or deletion. |
| **Redo Last Action** | `E` | Redo the previously undone action. |

## App Interface & Settings

| Action | Default Keybinding | Description |
| :--- | :--- | :--- |
| **Open Folder Picker** | `O` | Open the current working directory. Use the **Open** button in the sidebar to open the native folder picker dialog instead. |
| **Focus Search Bar** | `I` | Focus the search input field to filter media files. |
| **Toggle Metadata Panel** | `M` | Expand or collapse the right-hand metadata inspector. |
| **Create Folder** | `C` | Create a new subdirectory inside the active folder. |

---

## Pinned Folders

Pinned folders are target destinations assigned quick shortcuts:

| Action | Default Keybinding | Description |
| :--- | :--- | :--- |
| **Move to Pinned Folder 1-9** | `Alt + 1` to `Alt + 9` | Move selected media file directly to a pinned folder target. |
| **Open Folder Picker to Pin** | `P` | Open a native folder picker dialog to choose a new folder to pin. |
| **Pin Selected Folder** | `F` | Pin the selected folder in the folder tree. |
| **Unpin Current Folder** | `U` | Unpin the currently active folder. |
| **Move Pinned Up** | `Ctrl + W` | Reorder pinned folder list up. |
| **Move Pinned Down** | `Ctrl + S` | Reorder pinned folder list down. |

---

## Media Playback Shortcuts

These shortcuts are **hardcoded** and not configurable via the settings dialog:

| Action | Default Keybinding | Description |
| :--- | :--- | :--- |
| **Toggle Pause/Resume** | `Space` | Pause or resume video playback. |
| **Media Play/Pause** | `MediaPlayPause` / `MediaPlay` / `MediaPause` | Toggle video pause (multimedia keys). |
| **Stop Video** | `MediaStop` | Stop video playback. |
| **Next Media** | `MediaTrackNext` | Go to the next media file. |
| **Previous Media** | `MediaTrackPrevious` | Go to the previous media file. |
| **Volume Up** | `AudioVolumeUp` | Increase volume by 5%. |
| **Volume Down** | `AudioVolumeDown` | Decrease volume by 5%. |
| **Toggle Mute** | `AudioVolumeMute` | Toggle mute on/off. |
| **Quit Application** | `Ctrl + Q` | Exit Media Sort immediately. |
