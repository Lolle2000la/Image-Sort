Media Sort is designed from the ground up for maximum speed and efficiency using only your keyboard. This guide details the default navigational rules and keyboard mappings.

## Grid Navigation

Use the following keys to navigate the media grid (thumbnails):

| Action | Default Keybinding | Description |
| :--- | :--- | :--- |
| **Go Left / Previous** | <Kbd>Left</Kbd> | Select the previous media file. |
| **Go Right / Next** | <Kbd>Right</Kbd> | Select the next media file. |
| **Open Selected** | <Kbd>Enter</Kbd> | Open the selected folder or double-click behavior. |

---

## Folder Tree Navigation

Use the standard WASD keys to navigate, expand, and collapse the directory tree structure:

| Action | Default Keybinding | Description |
| :--- | :--- | :--- |
| **Move Selection Up** | <Kbd>W</Kbd> | Move the cursor up the folder tree. |
| **Move Selection Down** | <Kbd>S</Kbd> | Move the cursor down the folder tree. |
| **Collapse / Left** | <Kbd>A</Kbd> | Collapse the selected folder node or go to the parent node. |
| **Expand / Right** | <Kbd>D</Kbd> | Expand the selected folder node or go to its first child. |

---

## Sorting & Action Mappings

Perform file management tasks on the active media file instantly:

| Action | Default Keybinding | Description |
| :--- | :--- | :--- |
| **Move to Folder** | <Kbd>Up</Kbd> | Move the currently selected media file to the active folder. |
| **Copy to Folder** | <Kbd>Shift+Up</Kbd> | Copy the currently selected media file to the active folder. |
| **Delete** | <Kbd>Down</Kbd> | Move the currently selected media file to the system Trash. |
| **Rename** | <Kbd>R</Kbd> | Trigger the rename dialog for the current media file. |
| **Undo Last Action** | <Kbd>Q</Kbd> | Undo the last move, copy, rename, or deletion. |
| **Redo Last Action** | <Kbd>E</Kbd> | Redo the previously undone action. |

## App Interface & Settings

| Action | Default Keybinding | Description |
| :--- | :--- | :--- |
| **Open Folder Picker** | <Kbd>O</Kbd> | Open the native system dialog to select a new root directory. |
| **Focus Search Bar** | <Kbd>I</Kbd> | Focus the search input field to filter media files. |
| **Toggle Metadata Panel** | <Kbd>M</Kbd> | Expand or collapse the right-hand metadata inspector. |
| **Create Folder** | <Kbd>C</Kbd> | Create a new subdirectory inside the active folder. |

---

## Pinned Folders

Pinned folders are target destinations assigned quick shortcuts:

| Action | Default Keybinding | Description |
| :--- | :--- | :--- |
| **Move to Pinned Folder 1-9** | <Kbd>Alt+1</Kbd> to <Kbd>Alt+9</Kbd> | Move selected media file directly to a pinned folder target. |
| **Open Folder Picker to Pin** | <Kbd>P</Kbd> | Open a native folder picker dialog to choose a new folder to pin. |
| **Pin Selected Folder** | <Kbd>F</Kbd> | Pin the selected folder in the folder tree. |
| **Unpin Current Folder** | <Kbd>U</Kbd> | Unpin the currently active folder. |
| **Move Pinned Up** | <Kbd>Ctrl+W</Kbd> | Reorder pinned folder list up. |
| **Move Pinned Down** | <Kbd>Ctrl+S</Kbd> | Reorder pinned folder list down. |

---

## Media Playback Shortcuts

These shortcuts are **hardcoded** and not configurable via the settings dialog:

| Action | Default Keybinding | Description |
| :--- | :--- | :--- |
| **Toggle Pause/Resume** | <Kbd>Space</Kbd> | Pause or resume video playback. |
| **Media Play/Pause** | <Kbd>MediaPlayPause</Kbd> / <Kbd>MediaPlay</Kbd> / <Kbd>MediaPause</Kbd> | Toggle video pause (multimedia keys). |
| **Stop Video** | <Kbd>MediaStop</Kbd> | Stop video playback. |
| **Next Media** | <Kbd>MediaTrackNext</Kbd> | Go to the next media file. |
| **Previous Media** | <Kbd>MediaTrackPrevious</Kbd> | Go to the previous media file. |
| **Volume Up** | <Kbd>AudioVolumeUp</Kbd> | Increase volume by 5%. |
| **Volume Down** | <Kbd>AudioVolumeDown</Kbd> | Decrease volume by 5%. |
| **Toggle Mute** | <Kbd>AudioVolumeMute</Kbd> | Toggle mute on/off. |
| **Quit Application** | <Kbd>Ctrl+Q</Kbd> | Exit Media Sort immediately. |
