Media Sort allows you to customize all keyboard shortcuts. You can easily remap them directly in the application GUI or modify the raw configuration file for advanced needs.

---

## Remapping Keybindings via GUI

To customize a keyboard shortcut inside the app:

1. Click the **Open** button under **Settings** in the left sidebar panel (or press `Ctrl + ,`).
2. Select the **Key bindings** tab at the top of the settings window.
3. Click the secondary button showing the current keybinding next to the action you want to modify. The button text will change to *Press a key...*.
4. Press the key combination you wish to bind (including modifiers like `Ctrl`, `Shift`, or `Alt`).
5. Click **Close** to save your preferences.
