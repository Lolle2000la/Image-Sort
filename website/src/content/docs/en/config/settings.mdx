Media Sort provides a built-in Settings dialog that lets you customize the appearance, behavior, language, and shortcut keys of the application.

To open the settings dialog, click the **Open** button under **Settings** in the left sidebar navigation panel.

---

## General Settings

The **General settings** tab covers configuration options for directory behavior, updates, theme customisation, and operating system integrations.

### Appearance

#### Theme

Media Sort includes pre-configured visual themes to customize the interface. Selecting a theme immediately updates the colors of all UI components:

- Available options: **Dark**, **Light**, **Dracula**, **Nord**, **Solarized (Light/Dark)**, **Gruvbox (Light/Dark)**, **Catppuccin (Latte/Frappe/Macchiato/Mocha)**, **Tokyo Night**, **Kanagawa**, **Moonfly**, **Nightfly**, **Oxocarbon**, **Ferra**, and more.

#### Reopen last opened folder on startup

- **Behavior:** When enabled, the application automatically re-opens the directory you were working on when you last closed the application. If disabled, Media Sort starts with your system Pictures directory.

---

### Animated GIFs

#### Play animated gifs

- **Behavior:** Toggles whether GIF animations are active in the previewer and image grid panels. Disabling this displays a static first frame, which can optimize rendering performance on folder loads. A single toggle controls both the preview and grid behavior.

---

### Updates

:::note
Update features are supported on platforms built with Velopack bundle targets (currently Linux AppImage, Windows, and macOS packages).
:::

#### Check for updates on startup

- **Behavior:** Automatically checks the GitHub repository releases upon launching the application. If a new version is discovered, a dialog prompts you to apply the update.

#### Install prerelease builds

- **Behavior:** Includes pre-release (beta/alpha) builds in the update checker. Enable this option if you want to test and participate in verifying experimental features.

---

### Integration with Windows (Windows Only)

#### Explorer Context Menu

- **Behavior:** Integrates a context menu item into Windows Explorer. When right-clicking any folder in Windows, you can choose "Open in Media Sort" to launch the program directly in that workspace.

---

### Language

#### Language Selection

- **Behavior:** Allows you to select the interface language from all supported localizations. By default, Media Sort auto-detects your operating system's active language at launch and falls back to English if the system language is not supported.

---

## Keybindings

The **Key bindings** tab lets you remap all keyboard shortcuts. See the [Custom Keybindings](/config/keybindings/) page for details on how to use the GUI remapping editor and the [Configuration File](/advanced/config-file/) page for the TOML schema.

---

## Undo & Redo

Media Sort uses non-destructive file operations to support instant undo and redo of all sorting actions. Deleted files are moved to your system trash, not permanently removed. See the [Undo & Redo](/manuals/undo-redo/) page for a full explanation of how the history system works, including trash behavior and restoration limits.
