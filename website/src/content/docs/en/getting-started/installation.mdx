## 🛠️ System Prerequisites

To run Media Sort, you must have the `mpv` demuxer libraries (`libmpv`) installed on your system.

### Linux

#### Arch Linux / CachyOS
```bash
sudo pacman -S mpv
```

#### Debian / Ubuntu
```bash
sudo apt install libmpv-dev
```

#### Fedora
```bash
sudo dnf install mpv-libs-devel
```

### macOS

Ensure you have Homebrew installed, then run:
```bash
brew install mpv
```

### Windows
No manual setup is needed! The precompiled Windows installers and ZIP packages bundle the required `libmpv-2.dll` library natively.
