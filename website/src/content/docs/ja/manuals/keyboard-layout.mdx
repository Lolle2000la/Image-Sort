Media Sortは、キーボードのみで最大限の速度と効率を発揮できるよう設計されています。このガイドでは、デフォルトのナビゲーションルールとキー割り当てについて詳しく説明します。

## グリッドナビゲーション（メディアグリッド）

以下のキーを使用して、メディアグリッド（サムネイル一覧）内を移動します。

| アクション                  | デフォルトキーバインド | 説明                                                           |
| :-------------------------- | :--------------------- | :------------------------------------------------------------- |
| **左へ移動 / 前のファイル** | <Kbd>Left</Kbd>        | 前のメディアファイルを選択します。                             |
| **右へ移動 / 次のファイル** | <Kbd>Right</Kbd>       | 次のメディアファイルを選択します。                             |
| **選択した項目を開く**      | <Kbd>Enter</Kbd>       | 選択したフォルダを開くか、ダブルクリック時の動作を実行します。 |

---

## フォルダツリーナビゲーション

標準の WASD キーを使用して、ディレクトリツリー構造を移動、展開、折りたたみします。

| アクション            | デフォルトキーバインド | 説明                                                         |
| :-------------------- | :--------------------- | :----------------------------------------------------------- |
| **選択を上へ**        | <Kbd>W</Kbd>           | フォルダツリー上でカーソルを上に移動します。                 |
| **選択を下へ**        | <Kbd>S</Kbd>           | フォルダツリー上でカーソルを下に移動します。                 |
| **折りたたむ / 左へ** | <Kbd>A</Kbd>           | 選択したフォルダを折りたたむか、親フォルダに移動します。     |
| **展開する / 右へ**   | <Kbd>D</Kbd>           | 選択したフォルダを展開するか、最初の子フォルダに移動します。 |

---

## 整理＆ファイル管理アクション

選択中のメディアファイルに対して、即座にファイル操作を実行できます。

| アクション           | デフォルトキーバインド | 説明                                                           |
| :------------------- | :--------------------- | :------------------------------------------------------------- |
| **フォルダへ移動**   | <Kbd>Up</Kbd>          | 選択中のメディアファイルをアクティブなフォルダに移動します。   |
| **フォルダへコピー** | <Kbd>Shift+Up</Kbd>    | 選択中のメディアファイルをアクティブなフォルダにコピーします。 |
| **削除**             | <Kbd>Down</Kbd>        | 選択中のメディアファイルをシステムのゴミ箱に移動します。       |
| **名前変更**         | <Kbd>R</Kbd>           | 選択中のファイル名変更ダイアログを開きます。                   |
| **操作を取り消す**   | <Kbd>Q</Kbd>           | 直前の移動、コピー、名前変更、削除操作を取り消します（Undo）。 |
| **操作をやり直す**   | <Kbd>E</Kbd>           | 取り消した操作をやり直します（Redo）。                         |

## アプリケーションUI＆設定

| アクション                     | デフォルトキーバインド | 説明                                                               |
| :----------------------------- | :--------------------- | :----------------------------------------------------------------- |
| **フォルダを開く**             | <Kbd>O</Kbd>           | ルートディレクトリを選択するためのネイティブダイアログを開きます。 |
| **検索バーにフォーカス**       | <Kbd>I</Kbd>           | メディアファイルをフィルタするための検索入力欄にフォーカスします。 |
| **メタデータパネルの表示切替** | <Kbd>M</Kbd>           | 右側のメタデータインスペクタパネルを開閉します。                   |
| **フォルダ作成**               | <Kbd>C</Kbd>           | アクティブなフォルダ内に新しいサブディレクトリを作成します。       |

## ピン留めフォルダ

ピン留めフォルダは、クイックショートカットが割り当てられた整理先ターゲットです。

| アクション                           | デフォルトキーバインド               | 説明                                                                               |
| :----------------------------------- | :----------------------------------- | :--------------------------------------------------------------------------------- |
| **ピン留めフォルダ 1-9 へ移動**      | <Kbd>Alt+1</Kbd> 〜 <Kbd>Alt+9</Kbd> | 選択したメディアファイルをピン留めフォルダに直接移動します。                       |
| **フォルダピッカーを開いてピン留め** | <Kbd>P</Kbd>                         | ネイティブのフォルダ選択ダイアログを開き、ピン留めする新しいフォルダを選択します。 |
| **選択したフォルダをピン留め**       | <Kbd>F</Kbd>                         | フォルダツリーで選択されているフォルダをピン留めします。                           |
| **現在のフォルダのピン留めを解除**   | <Kbd>U</Kbd>                         | 現在アクティブなフォルダのピン留めを解除します。                                   |
| **ピン留めを上に移動**               | <Kbd>Ctrl+W</Kbd>                    | ピン留めフォルダのリスト表示順を上に移動します。                                   |
| **ピン留めを下に移動**               | <Kbd>Ctrl+S</Kbd>                    | ピン留めフォルダのリスト表示順を下に移動します。                                   |

## メディア再生ショートカット

以下のショートカットは**ハードコード**されており、設定ダイアログから変更することはできません。

| アクション                 | デフォルトキーバインド                                                   | 説明                                                 |
| :------------------------- | :----------------------------------------------------------------------- | :--------------------------------------------------- |
| **一時停止/再開の切替**    | <Kbd>Space</Kbd>                                                         | 動画の再生を一時停止または再開します。               |
| **メディア再生/一時停止**  | <Kbd>MediaPlayPause</Kbd> / <Kbd>MediaPlay</Kbd> / <Kbd>MediaPause</Kbd> | 動画の一時停止を切り替えます（マルチメディアキー）。 |
| **動画を停止**             | <Kbd>MediaStop</Kbd>                                                     | 動画の再生を停止します。                             |
| **次のメディア**           | <Kbd>MediaTrackNext</Kbd>                                                | 次のメディアファイルに移動します。                   |
| **前のメディア**           | <Kbd>MediaTrackPrevious</Kbd>                                            | 前のメディアファイルに移動します。                   |
| **音量を上げる**           | <Kbd>AudioVolumeUp</Kbd>                                                 | 音量を5%上げます。                                   |
| **音量を下げる**           | <Kbd>AudioVolumeDown</Kbd>                                               | 音量を5%下げます。                                   |
| **ミュート切替**           | <Kbd>AudioVolumeMute</Kbd>                                               | ミュートのオン/オフを切り替えます。                  |
| **アプリケーションを終了** | <Kbd>Ctrl+Q</Kbd>                                                        | Media Sortを即座に終了します。                       |
