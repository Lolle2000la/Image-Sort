---
title: キーボードレイアウトと操作方法
description: Media Sortのデフォルトキーバインドと操作ルール。
---

Media Sortは、キーボードのみで最大限の速度と効率を発揮できるよう設計されています。このガイドでは、デフォルトのナビゲーションルールとキー割り当てについて詳しく説明します。

## グリッドナビゲーション（メディアグリッド）

以下のキーを使用して、メディアグリッド（サムネイル一覧）内を移動します。

| アクション | デフォルトキーバインド | 説明 |
| :--- | :--- | :--- |
| **左へ移動 / 前のファイル** | `左矢印キー (Left Arrow)` | 前のメディアファイルを選択します。 |
| **右へ移動 / 次のファイル** | `右矢印キー (Right Arrow)` | 次のメディアファイルを選択します。 |
| **選択した項目を開く** | `Enterキー` | 選択したフォルダを開くか、ダブルクリック時の動作を実行します。 |

---

## フォルダツリーナビゲーション

標準の WASD キーを使用して、ディレクトリツリー構造を移動、展開、折りたたみします。

| アクション | デフォルトキーバインド | 説明 |
| :--- | :--- | :--- |
| **選択を上へ** | `W` | フォルダツリー上でカーソルを上に移動します。 |
| **選択を下へ** | `S` | フォルダツリー上でカーソルを下に移動します。 |
| **折りたたむ / 左へ** | `A` | 選択したフォルダを折りたたむか、親フォルダに移動します。 |
| **展開する / 右へ** | `D` | 選択したフォルダを展開するか、最初の子フォルダに移動します。 |

---

## 整理＆ファイル管理アクション

選択中のメディアファイルに対して、即座にファイル操作を実行できます。

| アクション | デフォルトキーバインド | 説明 |
| :--- | :--- | :--- |
| **フォルダへ移動** | `上矢印キー (Up Arrow)` | 選択中のメディアファイルをアクティブなフォルダに移動します。 |
| **フォルダへコピー** | `Shift + 上矢印キー` | 選択中のメディアファイルをアクティブなフォルダにコピーします。 |
| **削除** | `下矢印キー (Down Arrow)` | 選択中のメディアファイルをシステムのゴミ箱に移動します。 |
| **名前変更** | `R` | 選択中のファイル名変更ダイアログを開きます。 |
| **操作を取り消す** | `Q` | 直前の移動、コピー、名前変更、削除操作を取り消します（Undo）。 |
| **操作をやり直す** | `E` | 取り消した操作をやり直します（Redo）。 |

## アプリケーションUI＆設定

| アクション | デフォルトキーバインド | 説明 |
| :--- | :--- | :--- |
| **フォルダ選択を開く** | `O` | ルートディレクトリを選択するためのネイティブダイアログを開きます。 |
| **検索バーにフォーカス** | `I` | メディアファイルをフィルタするための検索入力欄にフォーカスします。 |
| **メタデータパネルの表示切替** | `M` | 右側のメタデータインスペクタパネルを開閉します。 |
| **フォルダ作成** | `C` | アクティブなフォルダ内に新しいサブディレクトリを作成します。 |

## ピン留めフォルダ

ピン留めフォルダは、クイックショートカットが割り当てられた整理先ターゲットです。

| アクション | デフォルトキーバインド | 説明 |
| :--- | :--- | :--- |
| **ピン留めフォルダ 1-9 へ移動** | `Alt + 1` 〜 `Alt + 9` | 選択したメディアファイルをピン留めフォルダに直接移動します。 |
| **アクティブフォルダをピン留め** | `P` | 現在アクティブなフォルダツリーノードをピン留めします。 |
| **選択したフォルダをピン留め** | `F` | 選択されているフォルダをピン留めします。 |
| **ピン留めを解除** | `U` | 選択されているフォルダのピン留めを解除します。 |
| **ピン留めを上に移動** | `Ctrl + W` | ピン留めフォルダのリスト表示順を上に移動します。 |
| **ピン留めを下に移動** | `Ctrl + S` | ピン留めフォルダのリスト表示順を下に移動します。 |
