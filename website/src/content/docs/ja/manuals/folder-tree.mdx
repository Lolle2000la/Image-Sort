Media Sortは、左パネルにファイルシステムをインタラクティブなフォルダツリーとして表示します。このツリーは現在開いているルートフォルダのディレクトリ構造を反映し、すべてキーボードでサブディレクトリを操作・管理できます。

## ツリーのナビゲーション

フォルダツリーは標準的なWASD方式のナビゲーションをサポートしています：

| キー         | 動作                                                                                             |
| :----------- | :----------------------------------------------------------------------------------------------- |
| <Kbd>W</Kbd> | 選択を前のフォルダに移動                                                                         |
| <Kbd>S</Kbd> | 選択を次のフォルダに移動                                                                         |
| <Kbd>A</Kbd> | 選択したフォルダを折りたたむ、または既に折りたたまれている場合は最も近い表示中の親フォルダに移動 |
| <Kbd>D</Kbd> | 選択したフォルダを展開する、または既に展開されている場合は最初の子フォルダに移動                 |

ツリーでフォルダを選択すると、メディアグリッドにはそのフォルダとそのサブディレクトリに含まれるすべてのファイルが再帰的に表示されます。

## 展開と折りたたみ

- **展開** (<Kbd>D</Kbd>): フォルダノードを開いて、その直下の子フォルダを一覧表示します（<span class="icon-inline"><ChevronDown size={14} /></span>アイコンで示されます）。サブフォルダはインデントされてネストの深さを示します。既に展開されている場合、<Kbd>D</Kbd>を押すと最初の子フォルダに選択が移動します。
- **折りたたみ** (<Kbd>A</Kbd>): 選択したノードの子フォルダを非表示にし、<span class="icon-inline"><ChevronRight size={14} /></span>アイコンに切り替わります。既に折りたたまれている場合、<Kbd>A</Kbd>を押すと最も近い表示中の親フォルダに選択が移動します。

## 親フォルダのパンくずリスト

現在のフォルダと各ピン留めフォルダには、サブフォルダ一覧のすぐ上に親ディレクトリの連鎖（パンくずリスト）が表示されます。これらのパンくずは完全なツリーノードであり、通常のフォルダアイコンが<span class="icon-inline"><ArrowUp size={14} /></span>アイコンに置き換えられ、通常のフォルダと同様に展開・折りたたみが可能で、独自のシェブロンインジケーターも表示されます。

たとえば、現在のフォルダが `/home/user/Pictures/Vacation/` の場合、ツリーは次のように表示されます：

```
📂 Vacation/                          （現在のフォルダ）
  ▲ home/                             （パンくず: クリックでそのディレクトリを開く）
    ▲ user/                           （パンくず）
      ▲ Pictures/                     （パンくず）
  📁 beach_photos/                     （実際のサブフォルダ）
  📁 mountains/                        （実際のサブフォルダ）
```

パンくずをクリックすると、そのディレクトリが新しい現在のフォルダとして開かれ、メディアグリッドとツリーが更新されます。

## フォルダを開く

選択したフォルダで <Kbd>Enter</Kbd> を押すと、そのフォルダが新しいルートフォルダとして開かれます。フォルダツリー、グリッド、検索がすべてリセットされ、新しく開いたディレクトリの内容が表示されます。

## 新しいフォルダを作成する

<Kbd>C</Kbd>
を押すと「フォルダを作成」ダイアログが開きます。新しいフォルダは現在選択されているフォルダのサブディレクトリとして作成されます。希望の名前を入力し、
<Kbd>Enter</Kbd> を押すか **作成** をクリックして確定します。

## フォルダアイコン

ツリーでは状態を表す3種類のLucideアイコンを使用します：

| アイコン                 | 意味                                                                 |
| :----------------------- | :------------------------------------------------------------------- |
| <Folder size={18} />     | フォルダは折りたたまれています。サブフォルダを含む可能性があります。 |
| <FolderOpen size={18} /> | フォルダは展開されています。子フォルダが表示されています。           |
| <ArrowUp size={18} />    | パンくず。祖先ディレクトリを開きます。                               |

## ピン留めフォルダ

ピン留めフォルダは、現在のフォルダの下に追加のルートレベルノードとして表示され、<span class="icon-inline"><Pin size={14} /></span>ピンアイコンと番号付きショートカットラベル（<Kbd>Alt+1</Kbd>〜<Kbd>Alt+9</Kbd>）が付きます。各ピン留めフォルダも展開時に独自のパンくずリストを表示します。詳細は[ピン留めフォルダ](/manuals/pinned-folders/)のページをご覧ください。

## スクロール

ディレクトリ構造が深いまたは広い場合、スクロールバーが自動的に表示されます。キーボードナビゲーション（<Kbd>W</Kbd> / <Kbd>S</Kbd>）により、選択したフォルダが表示されたままになるようツリーが自動的にスクロールします。
