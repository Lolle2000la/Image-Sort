メタデータパネルは、ウィンドウ右側にある折りたたみ可能なサイドバーで、現在選択中のメディアファイルの詳細情報を表示します。画像、音声ファイル、動画に対応し、各フォーマットが提供する最大限のメタデータを抽出します。

## 開閉

| キー         | アクション                              |
| :----------- | :-------------------------------------- |
| <Kbd>M</Kbd> | メタデータパネルの表示/非表示を切り替え |

左端をドラッグしてパネルの幅を変更することもできます。パネルの幅は設定に保存されます。

## メタデータセクション

パネルは折りたたみ可能なセクションで構成されています。すべてのセクションがすべてのファイルタイプで表示されるわけではありません。

### ファイル情報

すべてのメディアタイプで利用可能な基本ファイルプロパティ:

| フィールド     | 説明                                              |
| :------------- | :------------------------------------------------ |
| **Name**       | 拡張子付きのファイル名                            |
| **Size**       | ファイルサイズ（B、KB、MB、GBで自動フォーマット） |
| **Modified**   | 最終更新日時                                      |
| **Dimensions** | 画像/動画の幅×高さ（ピクセル）                    |
| **Duration**   | 音声・動画ファイルの再生時間                      |

### コンテナメタデータ

ファイルコンテナ形式に埋め込まれたタグ:

| フィールド       | 説明                                       |
| :--------------- | :----------------------------------------- |
| **Title**        | メディアタイトル                           |
| **Artist**       | 演奏アーティストまたは作者                 |
| **Album**        | アルバム名（音声）、コレクション名（動画） |
| **Genre**        | コンテンツのジャンル                       |
| **Date / Year**  | リリース日または録音日                     |
| **Track**        | トラック番号（音声）                       |
| **Comment**      | 埋め込みコメントまたは説明                 |
| **Copyright**    | 著作権表示                                 |
| **Language**     | コンテンツの言語コード                     |
| **Composer**     | 作曲者名                                   |
| **Conductor**    | 指揮者名                                   |
| **Lyrics**       | 埋め込み歌詞テキスト                       |
| **BPM**          | ビート毎分（音声）                         |
| **Label**        | レコードレーベル                           |
| **Disc Number**  | マルチディスクセットのディスク番号         |
| **Total Tracks** | アルバムの総トラック数                     |
| **Description**  | メディア説明                               |

### EXIFデータ（画像）

JPEG、TIFF、その他EXIF対応フォーマットから抽出された撮影メタデータ:

| フィールド          | 説明                              |
| :------------------ | :-------------------------------- |
| **Date/Time Taken** | 元の撮影日時                      |
| **Camera Maker**    | カメラメーカー（例: Canon、Sony） |
| **Camera Model**    | カメラモデル名                    |
| **Software**        | 画像処理ソフトウェア              |
| **Exposure Time**   | シャッター速度（例: 1/250）       |
| **Aperture**        | F値（例: f/2.8）                  |
| **ISO Speed**       | ISO感度                           |
| **Focal Length**    | レンズ焦点距離（mm）              |
| **Lens**            | レンズモデル名                    |

### GPSデータ（画像）

画像に位置情報のEXIFタグが含まれている場合:

| フィールド    | 説明             |
| :------------ | :--------------- |
| **Latitude**  | 緯度（10進数）   |
| **Longitude** | 経度（10進数）   |
| **Altitude**  | 標高（メートル） |

## フォーマット別対応

| メディアタイプ              | メタデータソース   | 使用ライブラリ          |
| :-------------------------- | :----------------- | :---------------------- |
| **画像**                    | EXIF、画像サイズ   | `kamadak-exif`、`image` |
| **音声（MP3/AIFF/WAV）**    | ID3タグ            | `id3`                   |
| **音声（FLAC）**            | Vorbisコメント     | `metaflac`              |
| **音声（M4A/AAC/ALAC）**    | MPEG-4コンテナタグ | `mp4ameta`              |
| **音声（OGG/Opus/WMA）**    | コンテナタグ       | `symphonia`             |
| **動画（MP4/M4V/MOV）**     | MPEG-4コンテナタグ | `mp4ameta`              |
| **動画（MKV/WebM/その他）** | コンテナタグ       | `symphonia`             |
