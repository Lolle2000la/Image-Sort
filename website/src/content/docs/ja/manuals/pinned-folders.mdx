ピン留めフォルダは、主要な整理先フォルダです。よく使う移動先フォルダをピン留めすることで、ワンキーでメディアファイルを移動またはコピーできます。

## ピン留めフォルダの仕組み

ピン留めフォルダはフォルダパネルの専用セクションに表示され、番号（1〜9）でラベル付けされます。各ピン留めフォルダには番号付きのショートカットが割り当てられます。

- <Kbd>Alt+1</Kbd>〜<Kbd>Alt+9</Kbd>:
  現在選択中のメディアファイルを対応するピン留めフォルダに直接移動します。
- <Kbd>Alt+Shift+1</Kbd>〜<Kbd>Alt+Shift+9</Kbd>:
  選択中のメディアファイルをピン留めフォルダにコピー（移動ではなく）します。

番号はピン留めフォルダのリスト内の順序によって決まります。

## フォルダのピン留め

フォルダをピン留めするには2つの方法があります。

| 方法                         | キー         | 説明                                                                                 |
| :--------------------------- | :----------- | :----------------------------------------------------------------------------------- |
| **選択フォルダをピン留め**   | <Kbd>F</Kbd> | フォルダツリーで現在ハイライトされているフォルダをピン留めします                     |
| **任意のフォルダをピン留め** | <Kbd>P</Kbd> | ネイティブのフォルダ選択ダイアログを開き、システム上の任意のディレクトリを選択します |

## ピン留めの解除

<Kbd>U</Kbd>
を押すと、ピン留めフォルダリストで現在選択中のフォルダのピン留めが解除されます。フォルダはピン留めリストから削除されますが、ディスクからは削除されません。

## ピン留めフォルダの並べ替え

ピン留めフォルダはリストの順序で番号付けされるため、並べ替えによって各番号がどのフォルダに割り当てられるかが変わります。

| キー              | アクション                                   |
| :---------------- | :------------------------------------------- |
| <Kbd>Ctrl+W</Kbd> | 選択したピン留めフォルダをリスト内で上に移動 |
| <Kbd>Ctrl+S</Kbd> | 選択したピン留めフォルダをリスト内で下に移動 |

最もよく使う移動先を1〜3番に配置するようにピン留めフォルダを並べ替えると、操作がより快適になります。

## 永続性

ピン留めフォルダは`config.toml`ファイルに保存され、アプリケーションの再起動後も維持されます。WPF v2.xアプリケーションからアップグレードした場合、自動レガシー設定移行にも含まれます。
