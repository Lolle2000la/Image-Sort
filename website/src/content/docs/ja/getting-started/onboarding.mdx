Media Sortを初めて起動すると、アプリケーションはメディアグリッドビューで直接開始します。ここでは、起動時の動作とワークスペースの設定方法を説明します。

## 起動時の動作

初回起動時、Media Sortは自動的にシステムのデフォルトの**ピクチャ**ディレクトリ（例: Linuxでは `~/Pictures`、macOSでは `~/Pictures`）を開きます。ファイルが存在しない場合、グリッドは空の状態になります。その場合はフォルダピッカーを使用して別のディレクトリを選択してください。2回目以降の起動時は、**最後に開いたフォルダを再表示**設定により、前回作業していたフォルダが自動的に再表示されます。

コマンドライン引数でディレクトリパスを渡して直接開くことも可能です：
```bash
media-sort-gui /path/to/your/media
```

以前に旧バージョンのWPF版（v2.x）を使用していた場合、Media Sortは古い `config.json` から新しい `config.toml` 形式へ、設定（ピン留めフォルダ、ダークモード、ホットキー、ウィンドウ位置）を**自動的に移行**します。手動での操作は不要です。

## 1. ルートディレクトリの選択

整理したいメディアファイル（画像、動画、音声）が含まれているディレクトリを選択するには：

- 左側のコントロールパネルにある**フォルダ**の下の**フォルダを開く**ボタンをクリックして、ネイティブのフォルダ選択ダイアログを開きます。
- または、<Kbd>O</Kbd> キーを押して、ネイティブのフォルダ選択ダイアログを開きます。
- <Kbd>Enter</Kbd> キーを押して、フォルダツリーで現在選択されているフォルダを開きます。
- Media Sortは選択したフォルダを自動的にスキャンし、サポートされているすべてのフォーマットを再帰的に検出して、構造化されたグリッドに表示します。

## 2. ピン留めフォルダの設定

ピン留めフォルダは、整理先となるターゲットフォルダです。フォルダをピン留めすることで、選択したメディアを1つのキー操作で移動できるショートカットが割り当てられます。

- 左側のフォルダツリーパネルを使用して、目的のディレクトリに移動します。
- <Kbd>F</Kbd> キーを押して選択したフォルダをピン留めするか、<Kbd>P</Kbd> キーを押してフォルダピッカーを開き、システム上の任意のディレクトリをピン留めします。
- ピン留めされると、これらのフォルダにはクイックキーショートカットが割り当てられます。<Kbd>Alt+1</Kbd> 〜 <Kbd>Alt+9</Kbd> を押して、選択したメディアファイルを対応するピン留めフォルダに直接移動します。
- <Kbd>Ctrl+W</Kbd> と <Kbd>Ctrl+S</Kbd> でピン留めフォルダの順序を並び替えられます。順序を変えると、各数字キーの割り当て先フォルダが変更されます。

## 3. セットアップのカスタマイズ

左側のコントロールパネルにある**設定**の下の**開く**ボタンをクリックして**設定**ダイアログを開き、以下を設定します：
- **テーマ**: Dark、Light、Dracula、Nord、Catppuccin などから選択
- **キーバインド**: すべてのショートカットをお好みのレイアウトに変更
- **言語**: 英語、ドイツ語、日本語を切り替え
- **GIFアニメーション**: グリッドとプレビューでのアニメーションGIF再生のオン/オフ
