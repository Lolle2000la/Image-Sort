Media Sortを初めて起動すると、アプリケーションはメディアグリッドビューで直接開始します。ここでは、起動時の動作とワークスペースの設定方法を説明します。

## 起動時の動作

初回起動時、Media Sortは自動的にシステムのデフォルトの**ピクチャ**ディレクトリ（例: Linuxでは `~/Pictures`、macOSでは `~/Pictures`）を開きます。ファイルが存在しない場合、グリッドは空の状態になります。その場合はフォルダピッカーを使用して別のディレクトリを選択してください。

以前に旧バージョンのWPF版（v2.x）を使用していた場合、Media Sortは古い `config.json` から新しい `config.toml` 形式へ、設定（ピン留めフォルダ、ダークモード、ホットキー、ウィンドウ位置）を**自動的に移行**します。手動での操作は不要です。

## 1. ルートディレクトリの選択

整理したいメディアファイル（画像、動画、音声）が含まれているディレクトリを選択するには：

- 左サイドバーの**フォルダ**の下にある**開く**ボタンをクリックして、ネイティブのフォルダ選択ダイアログを開きます。
- または、`O` キーを押して、ネイティブのフォルダ選択ダイアログを開きます。
- Media Sortは選択したフォルダを自動的にスキャンし、サポートされているすべてのフォーマットを再帰的に検出して、構造化されたグリッドに表示します。

## 2. ピン留めフォルダの設定

ピン留めフォルダは、整理先となるターゲットフォルダです。フォルダをピン留めすることで、選択したメディアを1つのキー操作で移動できるショートカットが割り当てられます。

- 左側のフォルダツリーパネルを使用して、目的のディレクトリに移動します。
- フォルダを選択して `F` キーを押してピン留めするか、ピン留めフォルダパネルを使用して整理先ディレクトリを追加します。
- ピン留めされると、これらのフォルダにはクイックキーショートカットが割り当てられます。`Alt + 1` 〜 `Alt + 9` を押して、選択したメディアファイルを対応するピン留めフォルダに直接移動します。

## 3. セットアップのカスタマイズ

左サイドバーの**設定**ダイアログを開いて以下を設定します：
- **テーマ** — Dark、Light、Dracula、Nord、Catppuccin などから選択
- **キーバインド** — すべてのショートカットをお好みのレイアウトに変更
- **言語** — 英語、ドイツ語、日本語を切り替え
- **GIFアニメーション** — グリッドとプレビューでのアニメーションGIF再生のオン/オフ
