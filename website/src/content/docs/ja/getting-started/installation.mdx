## システムの前提条件

お使いのプラットフォームによっては、`mpv` メディア再生エンジンのライブラリをインストールする必要があります：

### Windows & macOS
**手動のセットアップは不要です！** Windows版とmacOS版のプリコンパイルパッケージには、必要なすべての `libmpv` ライブラリがアプリケーション内に同梱されています。

### Linux
Linux版のAppImageパッケージは実行環境のシステムライブラリを利用するため、`libmpv` がインストールされている必要があります：

#### Arch Linux / CachyOS
```bash
sudo pacman -S mpv
```

#### Debian / Ubuntu
```bash
sudo apt install libmpv-dev
```

#### Fedora
```bash
sudo dnf install mpv-libs-devel
```
