Media Sortでは、すべてのキーボードショートカットをカスタマイズできます。アプリケーションのGUIから直接変更することも、より高度な設定として設定ファイルをテキストエディタで直接編集することも可能です。

## 利用可能なキーバインドアクション

Media Sortでは以下の設定可能なアクションが用意されています：

| カテゴリ | アクション |
| :--- | :--- |
| **画像管理** | 移動、コピー、削除、名前変更 |
| **画像ナビゲーション** | 左へ移動、右へ移動、画像検索、メタデータパネルの表示切替 |
| **フォルダツリー** | 選択を上/下へ移動、折りたたみ/展開、フォルダ作成、フォルダを開く、選択フォルダを開く |
| **ピン留めフォルダ** | アクティブフォルダをピン留め、選択項目をピン留め、ピン留め解除、並べ替え（上/下） |
| **履歴** | 元に戻す（Undo）、やり直す（Redo） |

## GUIからのキーバインド変更

アプリケーション内でショートカットを変更する手順は以下の通りです：

1. 左側のサイドバーパネルにある **設定** 項目の下にある **開く** ボタンをクリックします。
2. 設定ウィンドウの上部にある **キーバインド（Key bindings）** タブを選択します。
3. 変更したいアクションの右側にある、現在の割り当てが表示されたボタンをクリックします。ボタンのテキストが *キーを入力... (Press a key...)* に切り替わります。
4. 割り当てたいキーの組み合わせ（`Ctrl`、`Shift`、`Alt` などの修飾キーを含む）をキーボードで入力します。
5. **閉じる（Close）** をクリックして設定を保存します。

## 高度な設定: 設定ファイル

プログラムによる一括変更や詳細な設定には、`config.toml` ファイルを直接編集します。完全なスキーマ、キー名、および許可される値については、[設定ファイル](/advanced/config-file/)のページを参照してください。
