Media Sortでは、すべてのキーボードショートカットをカスタマイズできます。アプリケーションのGUIから直接変更することも、設定ファイルをテキストエディタで直接編集することも可能です。

---

## GUIからのキーバインド変更

アプリケーション内でショートカットを変更する手順は以下の通りです：

1. 左側のサイドバーにある **設定** 項目の下にある **開く** ボタンをクリックします（または `Ctrl + ,` キーを押します）。
2. 設定ウィンドウの上部にある **キーバインド（Keybindings）** タブを選択します。
3. 変更したいアクションの右側にある、現在の割り当てが表示されたボタンをクリックします。ボタンのテキストが *キーを入力... (Press a key...)* に切り替わります。
4. 割り当てたいキーの組み合わせ（`Ctrl`、`Shift`、`Alt` などの修飾キーを含む）をキーボードで入力します。
5. **閉じる（Close）** をクリックして設定を保存します。
